# Codigo de Conducta 📈

## 1. Proposito

El propósito de este Código de conducta es proporcionar pautas para los contribuyentes al proyecto Appdad en GitHub. Nuestro objetivo es crear un ambiente positivo e inclusivo donde todos los participantes puedan contribuir y colaborar de manera efectiva. Al participar en este proyecto, usted acepta cumplir con este Código de conducta.

## 2. Alcance

Este Código de conducta se aplica a todos los contribuyentes, mantenedores y usuarios del proyecto Appdad. Se extiende a todos los espacios del proyecto, incluidos, entre otros, problemas, solicitudes de extracción, revisiones de código, comentarios y otras formas de comunicación dentro del proyecto.

## 3. Nuestros estándares

Fomentamos el comportamiento respetuoso, considerado y colaborativo, buscando diversas perspectivas y ofreciendo comentarios constructivos y ayuda. Valoramos la empatía y la comprensión mutua.

Desalentamos el acoso, la discriminación y cualquier forma de lenguaje irrespetuoso, ofensivo o inapropiado. También desaprobamos los ataques personales, las críticas infundadas y la negatividad injustificada.

## 4. Informes y aplicación de la ley

Si es testigo o experimenta alguna violación de este Código de conducta, infórmelo a los encargados del proyecto por correo electrónico u otro medio apropiado. Los mantenedores investigarán y tomarán las medidas apropiadas, que pueden incluir advertencias, prohibiciones temporales o permanentes, u otras medidas según sea necesario.

Los mantenedores son responsables de garantizar el cumplimiento de este Código de conducta y pueden tomar medidas para abordar cualquier infracción.
